#  CricketVision Pro

**AI-Powered Cricket Performance Analytics Using Computer Vision and Data Science**

CricketVision Pro is a computer vision and data science project that analyzes cricket batting mechanics using a single webcam. The system detects human body pose in real time, collects custom batting datasets, and computes performance metrics that can help players understand and improve their technique.

This project combines computer vision, machine learning, data science, and software engineering into an end-to-end analytics platform.

---

##  Features

* Real-time human pose estimation using YOLO Pose
* Webcam-based batting session recording
* Automatic creation of custom motion datasets (CSV)
* Swing speed analysis
* Wrist movement visualization
* Data analysis with Pandas and NumPy
* Motion graphs using Matplotlib
* Version-controlled development with Git and GitHub

##  Current Workflow

1. Capture live video using a webcam.
2. Detect body keypoints using YOLO Pose.
3. Record batting sessions.
4. Store pose data in CSV format.
5. Analyze swing movement with Python.
6. Generate performance graphs and statistics.
